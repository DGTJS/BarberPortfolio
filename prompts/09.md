Em populares na @app\page.tsx vamos trocar para serviços que mais são agendamentos, quero uma lista no minimo 5 serviços

os 5 não precisam todos terem agendamentos porem os 5 devem ser os mais populares exemplo

1. Cabelo - 5 agendamentos
2. Barba - 3 agendamentos
3. acabamento - 2 agendamentos
4. Tratamento - 0 agendamento
5. Sobrancelha - 0 agendamento

caso não tenha agendamentos suficientes para adicionar os 5 serviços, adicione sempre por ordem alfabética

1. Cabelo
2. Barba
3. acabamento
4. Tratamento
5. Sobrancelha
